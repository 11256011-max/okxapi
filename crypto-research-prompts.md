## 安全提醒

- 不要把 API Key、Secret Key、Passphrase 貼到公開 repo 或聊天紀錄。
- 自動交易程式建議先從只讀行情開始，再到模擬盤，最後才小資金真實交易。
- API 權限建議只開讀取與交易，不要開提幣。
- 建議開啟 IP 白名單，並設定每日最大虧損、單筆最大風險與最大槓桿限制。
